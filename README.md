# Noster: Hope
A multiplayer game for two players. For connection to work, enable UPNP on your router.

## Tools used to create this game:

-   **Engine:**
    -   Godot v4.3
-   **Text Editor:**
    -   neovim
-   **3D models:**
    -   Blockbench
-   **Version Control:**
    -   git
    -   github
-   **Sounds & Music:**
    -   Musescore
    -   Furnace
    -   Audacity
